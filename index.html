FAQ

Frequently Asked Questions

Everything you need to know about the tool.

Do I need Python experience to use it?

Basic command-line familiarity is enough. Run python3 tool.py --data clients.csv and you get three reports. The sample data lets you try it in 30 seconds.

What data do I need to provide?

A CSV (or JSON) with client names, tier/plan level, monthly revenue, touch counts, and optional sentiment scores. A sample file with 12 accounts is included.

Can I customize the tier targets?

Absolutely. The tier configuration (touches, response time, channels) is defined at the top of the script — edit the Python dict and re-run.

Is this just for AI agencies?

No. The tool works for any B2B service business — marketing agencies, consulting firms, SaaS companies, freelancers with multiple clients.

How is this different from a CRM report?

CRMs show raw data. This tool scores, interprets, and prioritizes — telling you not just what's happening, but exactly what to do about each client.
